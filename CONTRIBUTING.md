# Contributing

Thanks for your interest in Globe.

## Development Setup

```sh
cd app
swift test
swift run Globe
```

## Guidelines

- Keep Globe native to macOS.
- Do not use private Apple APIs.
- Keep keyboard handling privacy-preserving.
- Add focused tests for behavior changes.
- Prefer small pull requests with clear motivation.

## Reporting Issues

Please include:

- macOS version
- Keyboard model
- Installed input sources involved
- Expected behavior
- Actual behavior
- Reproduction steps

# Security Policy

## Supported Versions

Globe is currently in public beta. Security fixes are released in the latest beta version only.

## Reporting a Vulnerability

Please do not open a public issue for a security vulnerability.

Report security concerns through:

- GitHub private vulnerability reporting, if available for this repository.
- Or contact Nythral through https://nythral.com.

Include:

- Globe version.
- macOS version.
- A clear description of the issue.
- Steps to reproduce.
- Whether the issue requires Accessibility permission.

## Privacy Boundary

Globe is designed not to record, store, or transmit typed text. Security reports involving keyboard handling, diagnostics, update downloads, signing, notarization, or Accessibility permission handling are especially useful.
